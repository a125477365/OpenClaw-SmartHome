## 七、接线颜色标准

```

┌────────────┬─────────────────────────────────────────┐
│ 线缆颜色   │ 用途                                   │
├────────────┼─────────────────────────────────────────┤
│ 🔴 红色    │ 火线 (L)                               │

- 不同地区可能有不同的颜色标准
```
