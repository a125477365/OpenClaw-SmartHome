# 智能开关硬件设计方案

## 设计目标

设计一款能够：
1. 与家用86型开关槽适配
2. 保留原有手动物理开关
3. 支持智能家居控制
4. 包含状态指示灯和初始化按钮
5. 支持传统手动开关方式（智能模块关闭时仍可手动控制）

## 关键设计：传统开关保留方案

### 方案：双控开关架构

```
原墙壁开关槽内的接线方式：

    火线(L) ─────────┬───────────────────────────────────────┐
                    │                                       │
                    ▼                                       │
            ┌───────────────┐                              │
            │   手动开关    │                              │
            │   (墙上原有)  │                              │
            └───────┬───────┘                              │
                    │                                       │
                    ▼                                       │
    ┌───────────────────────────────────────────────────────┤
    │                智能开关模块                            │
    │                                                       │
    │   ┌─────────┐      ┌─────────────┐                   │
    │   │ 继电器  │─────▶│ 输出(COM)   │───────▶ 灯具火线  │
    │   │ (常闭型)│      │             │                   │
    │   └─────────┘      └─────────────┘                   │
    │                                                       │
    └───────────────────────────────────────────────────────┘
                    │
                    ▼
                零线(N) ◀──────────────────────── 零线总线

关键点：
- 继电器使用"常闭型"(Normally Closed, NC)
- 智能模块断电/关闭时，继电器保持闭合
- 手动开关可以正常控制灯具
- 只有智能模块控制时，继电器才会断开（关灯）
```

### 工作模式说明

| 场景 | 手动开关状态 | 智能模块状态 | 继电器状态 | 灯具状态 |
|------|-------------|-------------|-----------|---------|
| 传统开灯 | 开 | 任意 | 闭合 | **亮** |
| 传统关灯 | 关 | 任意 | 闭合 | **灭** |
| 智能开灯 | 开 | 正常工作 | 闭合 | **亮** |
| 智能关灯 | 开 | 正常工作 | 断开 | **灭** |
| 模块断电 | 开 | 断电 | 闭合 | **亮**（传统控制正常） |
| 模块故障 | 任意 | 故障 | 闭合 | **手动开关正常工作** |

## PCB 布局设计

### 尺寸：86型开关槽适配

```
外形尺寸: 86mm x 86mm x 35mm (长x宽x高)
安装孔距: 60.3mm (标准86型)

┌─────────────────────────────────────────────────────────┐
│  ┌─────┐                                 ┌─────┐        │
│  │ LED │                                 │按钮 │        │
│  │状态 │                                 │重置 │        │
│  └─────┘                                 └─────┘        │
│                                                         │
│                    ┌─────────────────┐                  │
│                    │    ESP32模块    │                  │
│                    │   (天线朝外)    │                  │
│                    └─────────────────┘                  │
│                                                         │
│  ┌────────┐                              ┌────────┐     │
│  │ 电源   │                              │ 继电器 │     │
│  │ 模块   │                              │ 模块   │     │
│  └────────┘                              └────────┘     │
│                                                         │
│  ┌──────────────────────────────────────────────────┐   │
│  │              接线端子区                           │   │
│  │  L  N  COM  SW  │  L:火线  N:零线  COM:灯具  SW:开关│
│  └──────────────────────────────────────────────────┘   │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 布局要点

1. **强电弱电分离**：电源模块和继电器（强电区）与ESP32（弱电区）物理隔离
2. **爬电距离**：强弱电之间保持≥6mm爬电距离
3. **天线位置**：ESP32天线区域无铜皮覆盖，朝向外壳边缘
4. **散热考虑**：继电器和电源模块周围有散热孔设计

## 安装接线图

### 标准安装方式

```
墙壁开关槽内部接线:

┌─────────────────────────────────────────────────────────────────────┐
│                                                                     │
│   ┌─────────────────┐                                               │
│   │   原有开关      │                                               │
│   │   (保留)        │                                               │
│   └────────┬────────┘                                               │
│            │                                                         │
│            ▼                                                         │
│   ┌─────────────────────────────────────────────────────────────┐   │
│   │                    智能开关模块                              │   │
│   │                                                             │   │
│   │   ┌─────┐  ┌─────┐  ┌─────┐  ┌─────┐                       │   │
│   │   │ L   │  │ N   │  │COM  │  │ SW  │                       │   │
│   │   │火线 │  │零线 │  │输出 │  │开关 │                       │   │
│   │   └──┬──┘  └──┬──┘  └──┬──┘  └──┬──┘                       │   │
│   │      │        │        │        │                           │   │
│   └──────┼────────┼────────┼────────┼───────────────────────────┘   │
│          │        │        │        │                               │
│          │        │        │        │                               │
│          ▼        ▼        ▼        ▼                               │
│       火线总线   零线总线   灯具     开关线                         │
│       (进线)    (进线)     (去灯具) (从开关来)                      │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 接线步骤

1. **断电**：关闭总电源
2. **识别线路**：
   - L (火线)：通常为红色/棕色
   - N (零线)：通常为蓝色
   - 开关线：从原开关出来的线
   - 灯具线：去往灯具的线
3. **接线**：
   - 火线总线 → 智能模块 L 端子
   - 零线总线 → 智能模块 N 端子
   - 开关线 → 智能模块 SW 端子
   - 灯具线 → 智能模块 COM 端子
4. **固定模块**：将智能模块放入开关槽内
5. **恢复面板**：装回原有开关面板（或新面板）
6. **通电测试**：恢复电源，检查状态灯

## 安全设计

### 电气安全

1. **保险丝保护**：输入端250V/1A保险丝
2. **光电隔离**：继电器使用光耦隔离
3. **爬电距离**：强弱电之间≥6mm
4. **绝缘等级**：使用UL认证材料

### 认证要求

- 中国：CCC认证
- 欧洲：CE认证
- 国际：UL认证（可选）
