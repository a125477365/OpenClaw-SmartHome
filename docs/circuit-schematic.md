```
接线端子定义

│                                                                 │
│  端子排列 (从左到右)：                                          │
│                                                                 │
│  ┌────┬────┬────┬────┐                                          │

│  ┌────────────────────────────────────────────────────────────┐│

│  │  同时连接到灯具零线                                         ││

│                                                                 │
│  COM (输出到灯具)：                                             │
│  ┌────────────────────────────────────────────────────────────┐│

│                                                                 │

│  ┌────────────────────────────────────────────────────────────┐│
│  │  连接传统墙壁开关                                            ││
│  │  当传统开关闭合时，信号传给ESP32                            ││

│  └────────────────────────────────────────────────────────────┘│
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 五、给PCB设计师的说明

### 设计要点

1. **安全间距**
   - 强电区域（继电器触点、电源模块、接线端子）与弱电区域（ESP32）之间爬电距离≥6mm
   - 高压走线与低压走线分开区域

2. **走线宽度**
   - 电源线（+5V, GND）：≥0.5mm（或20mil）
   - 信号线（GPIO）：≥0.25mm（或10mil）
   - 强电线（AC）：≥1mm（或40mil）

3. **接地**
   - GND铺铜，增强抗干扰
   - 继电器线圈GND与ESP32 GND共地

4. **天线区域**
   - ESP32天线下方不要铺铜
   - 保持天线区域空旷

5. **过孔**
   - 电源孔：0.8mm
   - 信号孔：0.4mm
   - 强电孔：1.0mm以上
