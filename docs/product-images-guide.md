# 智能开关实物图 - 设计描述与生成提示词

## 一、产品渲染图描述

### 1.1 产品外观描述

```
产品名称：OpenClaw智能开关
产品类型：86型墙壁开关（智能家居）

整体外观：
- 形状：标准86型方盒，圆角设计
- 尺寸：86mm × 86mm × 35mm（长×宽×高）
- 颜色：哑光白色（PC阻燃材料）
- 表面：细磨砂质感，防指纹

面板设计：
- 顶部中央：小型圆形LED状态指示灯（直径3mm）
  - 配有透明亚克力灯罩，发光均匀
  - 颜色：RGB可变（蓝/绿/黄/红）
  
- 底部中央：圆形配网按钮（直径8mm）
  - 凹陷设计，防止误触
  - 按钮材质：与面板同色
  - 按钮周围有细微的凹槽装饰

侧面：
- 无多余凸起，整体平滑
- 顶部有细长散热孔（可选）
- 底部有接线端子区域

背面：
- 4个接线端子（L/N/COM/SW）
- 清晰的丝印标识
- 安装孔2个（对角位置）
```

### 1.2 产品材质与质感

```
材质描述：
- 面板：PC（聚碳酸酯）阻燃材料
- 颜色：RAL 9003信号白（温润白色）
- 表面处理：哑光磨砂，细颗粒感
- 触感：温润、不滑手、无指纹残留

质感参考：
- 类似Apple充电器的哑光质感
- 高端墙壁开关的细腻感
- 非廉价塑料的光滑感
```
