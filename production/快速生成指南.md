# OpenClaw 智能开关 - 一键生成指南

## 🎯 懒人方案：找淘宝代工

**最快方式：直接把项目文件发给淘宝店铺，让他们完成所有工作**

### 步骤 1：准备好文件包

```
发给淘宝的文件：
├── BOM.csv                    ← 元件清单
├── circuit-schematic.md       ← 电路原理图描述
├── pcb-design.json            ← PCB规格
├── enclosure-spec.md          ← 外壳规格
├── product-spec.json          ← 产品参数
└── 测试标准.md                ← 质检要求
```

### 步骤 2：找店铺

在淘宝搜索：

| 服务 | 搜索关键词 | 预估价格 | 交期 |
|------|----------|---------|------|
| PCB设计 | "PCB画板代画" | 50-150元 | 1-3天 |
| PCB打样 | "PCB打样" | 10-30元/5片 | 2-3天 |
| 外壳3D打印 | "3D打印服务" | 50-100元/套 | 3-5天 |
| 外壳开模 | "注塑开模" | 5000-20000元 | 15-30天 |
| SMT贴片 | "PCBA贴片" | 10-30元/片 | 3-5天 |

### 步骤 3：发送需求模板

```
【需求】智能开关PCB设计 + 打样

您好，需要设计一款智能开关PCB，规格如下：

1. 板尺寸：86mm x 86mm
2. 层数：2层
3. 主要元件：ESP32、继电器、电源模块
4. 特殊要求：强弱电分离，安全间距≥6mm

附件包含：
- 电路原理图描述
- 元件清单
- PCB规格参数

请报价和交期，谢谢！

联系人：[你的名字]
微信：[你的微信]
```

## 🚀 方案 B：自己用 EasyEDA（免费）

### 步骤 1：打开 EasyEDA

访问：https://lceda.cn/ （立创EDA，中文版）

### 步骤 2：创建项目

1. 注册登录
2. 点击「新建项目」→「OpenClaw-SmartSwitch」
3. 选择「新建原理图」

### 步骤 3：放置元件（按以下顺序）

```
搜索并放置：

1. 搜索 "ESP32" → 选择 "ESP32-WROOM-32D"
2. 搜索 "HLK-PM01" → 选择电源模块
3. 搜索 "Relay" → 选择 "SRD-05VDC"（继电器）
4. 搜索 "WS2812B" → 选择LED
5. 搜索 "S8050" → 选择三极管
6. 搜索 "1N4007" → 选择二极管
7. 搜索 "Capacitor" → 放置电容（100uF、0.1uF）
8. 搜索 "Resistor" → 放置电阻（1K、10K）
9. 搜索 "Button" → 放置按钮
10. 搜索 "Terminal" → 放置接线端子
```

### 步骤 4：连线

按照 `circuit-schematic.md` 的描述连接：

```
关键连接：
- ESP32 GPIO12 → R1(1K) → Q1基极 → 继电器线圈
- ESP32 GPIO13 → 按钮 → GND
- ESP32 GPIO14 → WS2812B DIN
- HLK-PM01 +5V → ESP32 VIN、继电器、LED VCC
- GND → 公共地
```

### 步骤 5：转PCB

1. 点击「设计」→「转换原理图到PCB」
2. 确认元件数量

### 步骤 6：PCB布局

按 `pcb-design.json` 的位置放置：

```
左边（弱电区）：
- ESP32模块：中心位置
- WS2812B LED：左上角
- 按钮：左下角

右边（强电区）：
- HLK-PM01：右上角
- 继电器：右下角
- 接线端子：右侧边缘

安全分隔线：X=45mm（中点）
```

### 步骤 7：布线

1. 点击「布线」→「自动布线」
2. 检查强弱电间距
3. 手动调整不符合要求的线

### 步骤 8：生成Gerber

1. 点击「文件」→「导出」→「Gerber」
2. 选择「生成所有层」
3. 下载 gerber.zip

### 步骤 9：下单打样

1. 点击「PCB下单」
2. 参数：
   - 尺寸：86mm x 86mm
   - 层数：2
   - 板厚：1.6mm
   - 数量：5片
   - 价格：约10元
3. 付款等待发货

## 📦 方案 C：直接买现成套件改装

**最简单：买现成的ESP32继电器模块**

### 推荐购买清单

```
淘宝搜索购买：

1. "ESP32继电器模块" - 约30-50元
   - 包含ESP32 + 继电器 + 电源

2. "86型开关面板空白" - 约5-10元
   - 用于安装模块

3. "WS2812B LED模块" - 约3元
   - 状态指示灯

4. "接线端子" - 约2元
   - 外部接线用

总成本：约40-65元
```

### 改装步骤

1. 把ESP32继电器模块装入86型外壳
2. 接线：L/N/COM/SW
3. 烧录固件
4. 测试功能
