## ⚠️ 重要提醒

### 1. 继电器一定要买常闭型！

```
正确型号：SRD-05VDC-SL-C (NC型)
错误型号：SRD-05VDC-SL-A (NO型)

区分方法：
- NC (Normally Closed) = 常闭型 = 断电时闭合 = 我们要的！
- NO (Normally Open) = 常开型 = 断电时断开 = 不对！

接线方法：
- COM：公共端
- NC：常闭端（我们要用的）
- NO：常开端（不用）

接线：
火线进 → COM
NC → 灯具火线
```

### 2. 电源模块推荐HLK-PM01

```
为什么不推荐廉价电源：
- 廉价电源纹波大，干扰ESP32 WiFi
- 安全性差，有起火风险
- HLK-PM01是工业级，稳定可靠

注意：有假货！建议买带包装的正品
```

### 3. ESP32天线选择

```
两种型号：
- ESP32-WROOM-32D：内置天线（推荐，简单）
- ESP32-WROOM-32E：外置天线（信号更好，需要接天线）

如果安装位置WiFi信号弱，选32E外接天线
```
