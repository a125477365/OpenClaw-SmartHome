## 设计规格

### PCB 尺寸
- **外形尺寸**: 86mm x 86mm (标准86型开关槽)
- **板厚**: 1.6mm
- **层数**: 2层
- **铜厚**: 1oz

### 安装孔
- **规格**: M3 (直径4mm)
- **位置**: (13, 13), (73, 13), (13, 73), (73, 73)
- **间距**: 60mm x 60mm (标准86型)

### 强弱电分隔
- **分隔线**: X = 45mm
- **安全间距**: >= 6mm
- **弱电区**: X < 45mm (ESP32、LED、按钮)
- **强电区**: X >= 45mm (电源、继电器、接线端子)

## 使用说明

### PCB 打样参数
```
尺寸: 86mm x 86mm
层数: 2层
板厚: 1.6mm
铜厚: 1oz
表面处理: HASL (有铅喷锡) 或 ENIG (沉金)
阻焊颜色: 绿色
丝印颜色: 白色
最小线宽: 0.2mm
最小间距: 0.2mm (强电区 >= 6mm)
```

### 打样数量建议
- 测试: 5片
- 小批量: 20-50片
- 量产: 100片起

## 安全提示

1. **强电区域严禁触摸** - 调试时必须断电
2. **爬电距离** - 强弱电之间保持 >= 6mm
3. **保险丝必须安装** - 过流保护
4. **外壳必须阻燃** - PC阻燃材料
